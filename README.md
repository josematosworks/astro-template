# Astro Template: Basics

A modern Astro template with React components, TailwindCSS, shadcn/ui components, and TypeScript support.

## üöÄ Features

- Astro with React integration
- TailwindCSS for styling
- shadcn/ui components
- TypeScript support
- ESLint & Prettier configuration
- Automatic formatting on save
- Lucide icons integration
- Form handling with react-hook-form
- Date handling with date-fns
- Carousel support with embla-carousel
- Charts with recharts
- Toast notifications with sonner

## üìÅ Project Structure

- `src/pages/` - Astro and Markdown files for page routing
- `src/components/` - UI components (Astro/React)
- `src/layouts/` - Page layouts
- `src/styles/` - Global styles and TailwindCSS configuration
- `public/` - Static assets

## üßû Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
| `npm run format`          | Format code with Prettier                        |
| `npm run lint`            | Lint and fix issues with ESLint                  |
| `npm run lint:check`      | Check for linting issues without fixing          |
